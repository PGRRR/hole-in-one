# ⛳ Hole in one 프로젝트

## 개요 📋

> 프로젝트 명 : 골프장 예약 서비스 (Hole in one)

### 🏗️ 개발 인원 및 기간
- 개발 기간 : 2022/05/20 ~ 2022/06/17 (4주간)
- 개발 인원 : Front-end 1명, Back-end 2명

- 가게 목록
- 예약
- 커뮤니티

> Back-end 
- Language : Java

- Database : MySQL
> Front-end
- Language : HTML5, CSS3, JavaScript
- Framework : React
